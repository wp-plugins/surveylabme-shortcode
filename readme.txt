SurveyLab.me Shortcode Plugin offers an easy way to allow users to integrate and embed social survey, opinions, polls and questions created with SurveyLab.me service.

== Description ==

This plugin will allow you to embed SurveyLab.me survey/question inside your post/page.

You can embed a survey/question with one of these ways:
    1) Paste directly the url of a survey/question in the post: https://surveylab.me/survey/nomination-oscar-2015-miglior-film-danimazione-468/
    2) Paste the short code `[surveylab id="SURVEYLAB-ID-HERE"]` where you'd like the embedded survey/question to appear. You can get the ID of your survey/question from the URL of the page when you create a survey/question. For instance, the ID of [this survey/question](https://surveylab.me/survey/nomination-oscar-2015-miglior-film-danimazione-468/) is "468".

== Installation ==

1. Install the SurveyLab plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the post or page where you\'d like to insert an embedded survey/question.
4. You can embed a survey/question with one of these ways:
    4a. Paste directly the url of a survey/question in the post: https://surveylab.me/survey/nomination-oscar-2015-miglior-film-danimazione-468/
    4b. Paste the short code `[surveylab id="SURVEYLAB-ID-HERE"]` where you'd like the embedded survey/question to appear. You can get the ID of your survey/question from the URL of the page when you create a survey/question. For instance, the ID of [this survey/question](https://surveylab.me/survey/nomination-oscar-2015-miglior-film-danimazione-468/) is "468".

== Frequently Asked Questions ==

= How to create a survey? =

You just need to go to https://surveylab.me/company/ and follow the simple wizard to create a survey in less than a minute!

= Can I embed more than a survey in the same post or page? =

Sure, as many as you want!

= Can I embed a survey inside a Widget? =

Sure, we already support this feature! Just simple past the survey\'s link or the shortcode.
